* Builds a `Spectre.Console` `Rows` output for an exception call tree by resolving traversal settings, choosing the effective root node/title, and composing a title `Markup` plus a rendered tree.
* Resolves which node should be treated as the displayed root by applying root filtering, runtime inclusion, and root mode through `ProfilerCallTreeRootResolver`.
* Creates the top-level tree node text with `ProfilerCallTreeFormatter.FormatExceptionCallTreeLine`, including root-specific formatting such as `isRoot: true` and an optional root label override.
* Recursively walks child `CallTreeNode`s, enforcing `MaxDepth`, adding formatted child nodes to `IHasTreeNodes`, and only descending when the child is not a special stop leaf and still has visible descendants.
* Classifies leaf rendering based on both `ShouldStopAtLeaf(GetCallTreeMatchName(child))` and whether filtered children remain, so displayed nodes reflect truncation and terminal nodes accurately.

`./ProfilerTreeFactory.cs`

* Accepts a `Theme` dependency and extracts its `TreeGuideColor` setting to configure how tree guide lines should be rendered.
* Converts the theme’s hex color string into a Spectre.Console color via `ConsoleThemeHelpers.ParseHexColor`, then wraps it in a reusable `Style` cached in a private field.
* Creates a new Spectre.Console `Tree` for a supplied root label instead of mutating an existing tree instance.
* Applies the cached guide-line style to each created tree by assigning the `Style` property during object initialization.

`./ProfilerTreeRootSelectionOptions.cs`

* Carries the input parameters that control how a profiler call tree root should be selected: optional root text filter, whether runtime frames are included, optional root-selection mode, and traversal limits.
* Persists traversal-shaping values as raw scalar options by holding `MaxDepth`, `MaxWidth`, and `SiblingCutoffPercent` together with the root-selection inputs in one immutable record instance.
* Converts its traversal-related fields into a `CallTreeTraversalSettings` object by calling `CallTreeTraversalSettings.Create(MaxDepth, MaxWidth, SiblingCutoffPercent)`.
